[Warframe]Equinox

2019-01-31

写在前面

Equinox是我使用率第三的战甲，原因也很简单，全自动杀怪太爽了XD，范围又大蓝耗又低，带个牛伤害也很够。外形我也挺喜欢，两个耳朵（应该是耳朵吧）和圆圆大大的头看起来也蛮可爱（我知道你们想说我审美DE化=。=）

Equinox这个名字我很喜欢，也很符合这个战甲的设计，标题没有用全大写名称也是因为Equinox写出来比较好看。

虽然Equinox外号是扶她（我平时也这么叫它），但是官方已经决定了扶她的两面形态都是女性，大家叫惯了比较不想改口吧。不过话说回来白面的形态真的很像汉子啊2333

EQUINOX | 扶她/阴阳甲

技能介绍

  • 1技能：在黑面和白面之间切换形态，不同的形态技能效果不同

    • 1技能集团强化卡很有意思，使用1技能时将召唤出另一面形态自动战斗，当年用来指数挂机炮台(你肯定能干掉不少人.mp3)

  • 2技能：

    • 黑面形态：使准星所指区域的敌人沉睡，范围不算太大
      • 怪睡着之后就可以暗杀了，有暗杀经验加成，这也是塞歼扶她水专精/互水甲的原理
    • 白面形态：激怒准星所指区域的敌人，被激怒的敌人会加速，同时变得更加脆弱
      • 加速和弱化都受技能强度影响

  • 3技能：

    • 黑面形态：降低附近的怪物的伤害
      • 怪物离扶她越近，伤害降得越多
      • 能力发动时，效果范围内的每个敌人都会产生1.5 / 1.25 / 1 / 0.5的每秒能量损耗。
    • 白面形态：提高附近友军(包括自己)的技能强度
      • 这个和距离没有关系，只要在技能范围内就能享受加成。
      • 被白面3技能影响的队友(包括自己)每次使用技能，都会消耗扶她的能量

  • 4技能：

    • 黑面形态：Equinox放出冥想光环，每个敌人在光环内被杀死都会恢复范围内的队友的护盾，此外光环还会积累范围内死亡的敌人50% / 55% / 60% / 75%的最大生命值。
      • 在技能停止时，积累的生命值会均分到所有范围内的队友身上来治疗队友。
    • 白面形态：Equinox释放出撕裂光环，对范围内的敌人造成切割伤害并造成出血效果。此外光环还会积累范围内死亡的敌人50% / 55% / 60% / 75%的最大生命值和护盾值。
      • 在技能停止时，积累的伤害值就会施加到范围内的所有敌人身上。
      • 进入光环的敌人会陷入眩晕状态。

配卡推荐

扶她现在的玩法还蛮多的。。。从IO刷核桃到塞歼互水还有辣椒生存和精英圣殿水经验都能看见。

主要分成两个大方向：输出型和能力型。基于这两种方向的配卡根据不同任务稍作调整即可

  • 主白面3和4的 输出扶她
  • 主黑面2的 睡眠扶她

这里是以赤毒生存的配卡为例，如果想打IO的话可以降低强度拉大范围，想刷精英圣殿的话可以再提高一点强度

这两个是输出扶她，操作方法是：切换到白面，先开3提高技能强度，再开4打输出。

如果有吼牛蛆爹配合的话一般是：白面扶她开3开4攒伤害把和平挑衅加成叠满->吼牛蛆爹准备好，然后站在扶她3范围里->蛆爹连吼牛，吼牛吼扶她->扶她跳崖卡吼->扶她开3开4输出，和平挑衅叠满之后重新开4刷新强度

效率一定要满，范围和强度视具体任务进行调整。

这是睡眠型扶她，这个打法操作比较简单，换到黑面222222就完事了。所以主效率主范围，持续不能短。
