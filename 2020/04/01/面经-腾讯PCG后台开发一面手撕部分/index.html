[面经]腾讯PCG后台开发一面手撕部分

2020-04-01

接上一篇[面经]腾讯PCG后台开发实习

和上一篇的电话面试同一天的晚上七点，面试官发的腾讯会议，进去共享屏幕写代码。

题目

题目是面试官发给我的，说我可以保存一下，我直接贴过来了：

  1. long a = (long)(((int *)0)+4);a 等于多少？
  2. int (*s[10])(int) 表示的是什么？

  3. 用熟悉的语言写一个函数，把字符串里面的空格全部去掉，并返回删除的空格的个数，要求空间复杂度 O(1)，函数原型：int delete_space(char *str)

  4. 编写一个函数求给定两个正整数的二进制表示中不同位数的个数。例如给定数字 8 和 9，8 的二进制表示是 1000，9 的二进制表示是 1001，则不同位数的个数为 1

  5. 编写一个函数将字符串转换为整数（即 atoi）

  6. 设计并实现一个 QQ 用户（QQ 用户 id 是一个 32 位整数）在线统计的 API，至少包含以下功能：

    用户上线登记，下线取消登记；快速判断某用户是否在线（高频调用接口）；快速统计当前多少用户在线；提供历史最高在线数的接口；统计最近 5 分钟内上线的用户数

体验

面试官人很好，比如我 atoi 的第一版实现太简单了，面试官就提醒我：先问了下我有没有看过库函数里的 atoi 的实现考虑了些什么问题。我说没看过，然后他就让我考虑一下各种情况，然后和我一起完善、讨论边界条件。

然后腾讯会议里面试官的头像是真人哈哈哈哈，面试官看起来是个很精神的开发，电话和会议里的语气也挺有活力也好相处，体验确实不错。
