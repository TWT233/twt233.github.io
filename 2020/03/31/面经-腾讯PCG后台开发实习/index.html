[面经]腾讯PCG后台开发实习

2020-03-31

写在前面

今天下午三点多的时候突击电话面试，第一次遇到这种突击面试，我有点紧张了，表现得不是太好。

面试

面试官先介绍了一下自己：PCG 后台开发，base 深圳，有意吗？有意的话我们现在就电话面试一会儿。

问项目

让我先讲讲自己做的开发项目。

问技术

C++

  • new 和 malloc 的区别
  • 指针和引用的区别
  • C++ 里有多态吗？怎么实现的

计网

  • TCP 和 UDP 的区别
  • TCP 连接建立过程，为什么要三次握手，而不是两次握手
  • HTTPS 了解过吗？讲讲和 HTTP 的区别
  • HTTPS 中，证书的验证是用什么样的方式

算法/数据结构

  • 讲讲快排是怎么样的一个过程
  • 快排的时间复杂度
    • 快排的最坏时间复杂度，什么情况，为什么
  • 题：32bit 的电脑上，有个文件中有 5e 个 int 型数据，求只出现过一次的数据。
    • 讲讲时间复杂度和空间复杂度

下期预告

电话打到后面，面试官说：这两天你什么时候有时间不？要做个笔试，我发腾讯会议给你。然后我说我都 OK，他就很果断地说：那今晚怎么样。我 OK 都说了我还能怎么办 hhh，那就今晚呗，然后就定了晚上七点开始。
